# OctoAcme Project Management Docs

## Summary
This README is the starting point for understanding OctoAcme's project management processes. It provides a concise summary of how projects are managed and links to all key process documents.

### Project Management Processes Overview

OctoAcme practices customer-first, iterative delivery with clear ownership and data-informed decisions. Project managers and product leads drive outcomes, supported by robust risk management, quality checks, and consistent communication. Project lifecycle phases include Initiation, Planning, Execution, Release, and Retrospective, all documented for transparent, repeatable project delivery.
